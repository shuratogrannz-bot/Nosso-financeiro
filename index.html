        Sincronização online (JSONBin)

        Cole aqui a chave e o Bin ID da sua conta em jsonbin.io. Esses dados ficam salvos apenas neste navegador e são usados para sincronizar os dados entre os dispositivos de vocês.

        Use uma Access Key individual (uma para cada pessoa, criada no jsonbin.io com permissão de leitura/escrita só neste bin) para poder revogar o acesso de uma pessoa sem afetar as outras. A Master Key dá acesso total à conta inteira e é compartilhada por todos que a usarem.
